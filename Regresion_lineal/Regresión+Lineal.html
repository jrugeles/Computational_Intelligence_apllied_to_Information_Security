# An√°lisis del modelo 
# En el modelo:
# y: es la respuesta
# Œ≤0: Es la constante de intersecci√≥n 
# Œ≤1: Es el coeficiente para x1 (primera caracter√≠stica..)
# Œ≤n: Es la caracter√≠stica n (enesima caracter√≠stica ..)

# Los valores Œ≤ son llamados los coeficientes del modelo
